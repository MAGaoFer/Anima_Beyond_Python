# Release Windows - v1.4.0

## ✅ Descarga rápida

Descarga: **AnimaBeyondFantasy_Windows_Portable.zip**

Pasos:
1. Descarga el ZIP.
2. Descomprímelo en una carpeta con permisos de escritura (por ejemplo, Escritorio o Documentos).
3. Ejecuta **AnimaBeyondFantasy.exe**.

No necesitas instalar Python ni ejecutar scripts de build.

## ℹ️ Dónde se guardan las fichas

Las fichas se guardan junto a la aplicación, en:

**Personajes/personajes/**

## 🆕 Novedades de esta versión

- Nueva propiedad **Natura** para PNJ en CLI y GUI.
- Los PNJ **sin Natura** no obtienen tiradas abiertas automaticas en iniciativa, ataque ni defensa.
- Los PJ ahora se comportan siempre con Natura activa (como hasta ahora).
- Compatibilidad hacia atras: PNJ antiguos sin campo `natura` siguen cargando con Natura activa.

## 🛠️ Notas conocidas

- No hay incidencias nuevas conocidas para esta version.

## 🔁 Actualizar desde una versión anterior

Si ya tenías fichas guardadas:
- Copia tu carpeta anterior **Personajes/personajes/** dentro de la nueva carpeta descomprimida.
- Sobrescribe cuando se solicite.
