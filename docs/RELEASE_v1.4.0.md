# Release v1.4.0 (2026-03-06)

## Descarga rapida

Descarga: **AnimaBeyondFantasy_Windows_Portable.zip**

Pasos:
1. Descarga el ZIP.
2. Descomprímelo en una carpeta con permisos de escritura (por ejemplo, Escritorio o Documentos).
3. Ejecuta **AnimaBeyondFantasy.exe**.

No necesitas instalar Python ni ejecutar scripts de build.

## Donde se guardan las fichas

Las fichas se guardan junto a la aplicacion, en:

**Personajes/personajes/**

## Novedades

- Nueva propiedad **Natura** para PNJ en CLI y GUI.
- Los PNJ **sin Natura** no obtienen tiradas abiertas automaticas en iniciativa, ataque ni defensa.
- Los PJ mantienen el comportamiento previo: siempre tienen Natura activa.
- Compatibilidad hacia atras: PNJ antiguos sin campo `natura` cargan con Natura activa.

## Notas

- Sin incidencias nuevas conocidas en esta version.
- Se recomienda hacer copia de seguridad de `Personajes/personajes/` antes de actualizar.
